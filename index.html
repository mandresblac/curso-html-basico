Etiquetas semanticas estructurales

Div es una etiqueta contenedora que no tiene semántica
header - Cabecera de un sitio Web o de una seccion
nav - Representa un navegacionb
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento
aside - Representa un contenido complementario o secundario, sirve para publicidad
section - Representa una seccion de contenido generico
article - Representa una seccion de autocontenido
footer- Representa el pie de pagina de un sitio web o de una seccion del sisio
address - Representa una informacion de contacto

Etiquetas de bloque Vs. etiquetas de linea

La etiqueta de bloque por exelencia es la etiqueta div, una etiqueta de bloque visualmente ocupa todo el espacio disponible en pantalla (El 100% del espacio (el ancho) disponible del viewport) y se posicionan una encima de otra, generan un salto de línea arriba y abajo
La etiqueta de línea por exelencia es la etiqueta span, una etiqueta de línea solo ocupa el espacio necesario que tiene su contenido, se ubican uno al lado del otro en la misma línea horizontal Hola soy otra span

Iframes

El elemento HTML iframe (de inline frame) representa un contexto de navegación anidado, el cual permite incrustrar otra página HTML en la página actual.

Los Iframes sirven para definir un contenedor para un contexto de navegación anidado :puede incluir una página web externa dentro de otra página web, un mapa de google maps, un video de youtube, un archivo en formato pdf, etc.
